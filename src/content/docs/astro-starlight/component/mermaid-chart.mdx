import MermaidChart from "@components/MermaidChart.astro";

## 概要

`<MermaidChart>` コンポーネントはコードブロックの記述を利用した、JavaScript 拡張の Mermaid グラフをレンダリングするコンポーネントです。

### インポート

```mdx
import Table from "@components/Table.astro";

<MermaidChart
    chart={`
    graph TD

    A((A サービス))

    subgraph child
      B[B サービス]
      C[C サービス]
      D[D サービス]
    end

    A -.-|B から取得 ⬆️| B
    A ---|C に渡す ⬇️| C
    A ---|D に渡す ⬇️| D

    `}

/>
```

#### プロパティ

##### chart

type: `string`

コードブロックに渡すテキスト
