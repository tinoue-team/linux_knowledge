## デフォルトのサイドバーの動き

Starlight はデフォルトで、ドキュメントのファイルシステム構造に基づいてサイドバーを自動的に生成する。
その際、各ファイルの `title` プロパティをサイドバーのエントリとして使用する。

## カスタム HTML 属性

ページのファイルの link 要素に attrs プロパティを使用すると

```js {10}
starlight({
    sidebar: [
        {
            label: "ガイド",
            items: [
                // 新しいタブで開かれる、Astroドキュメントへの外部リンク。
                {
                    label: "Astro Docs",
                    link: "https://docs.astro.build/",
                    attrs: { target: "_blank", style: "font-style: italic" },
                },
            ],
        },
    ],
});
```

-   `target: '_blank'`: リンクが新しいタブで開かれる
-   `style: 'プロパティ: 値'`: リンクのラベルにカスタムの style を指定
