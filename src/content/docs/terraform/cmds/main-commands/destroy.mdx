## 説明

### 解説

Terraform コマンドで管理しているインフラを削除する。

`terraform apply -destroy` のエイリアスコマンド

このコマンドは、`terraform plan` コマンドで受け付けるプランのカスタマイズオプションの多くも受け付ける。
これらのオプションの詳細については `terraform plan -help`

### 構文

```bash frame=none
terraform [global options] destroy [options]
```

## Sample

```bash frame="none"
terraform destroy
```

```bash frame="none"
terraform destroy -auto-approve
```
