## 説明

### 解説

Terraform ステートファイルから出力変数を読み込み、その値を表示する。引数を追加しない場合、output はルートモジュールのすべての出力を表示する。
NAME が指定されていない場合は、すべての出力が表示されます。

### 構文

```bash
terraform [global options] output [options] [NAME]
```
