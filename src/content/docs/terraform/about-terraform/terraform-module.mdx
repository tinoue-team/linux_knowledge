### module ファイルのディレクトリ管理

<FileTree>

    -   project/
        -   main.tf # terraform apply の対象となるファイル

        -   terraform.tfstate # 実行後に生成

### module セクションの書き方

main.tf 内で直接書いている場合

```tf
provider "aws" {
    region = "ap-northeast-1"
}

module "example_bucket" {
    source      = "./modules/s3_bucket"
    bucket_name = "example-bucket-20230519"
}
```
