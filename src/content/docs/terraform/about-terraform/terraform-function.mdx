### element

element 関数は、指定されたインデックスにあるリスト内の要素を返します。主にリストから値を取得する際に使用されます。この関数を空のリストで使用すると、エラーが発生します。

#### syntax

```tf
element(LIST, INDEX)
```

##### LIST

[required]  
要素を検索する対象のリスト。リストは、順序付けされた値のコレクション

##### INDEX

[required]  
LIST から取得したい要素のインデックス（0 から始まる整数）番号。
INDEX が LIST の範囲外の場合、エラー

#### sample

```tf
variable "example_list" {
    default = ["a", "b", "c"]
}

output "example_element" {
    value = element(var.example_list, 1)
}
```
