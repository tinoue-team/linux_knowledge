```
break [n]
              Exit from within a for, while, until, or select loop.  If n is specified, break n levels.  n must be ≥ 1.  If n is greater than the number of enclosing loops, all
              enclosing loops are exited.  The return value is non-zero when n is ≤ 0; Otherwise, break returns 0 value.
```
