## `ssh` コマンドの基本的な動作

ssh(SSH クライアント)はリモートマシンにログインし、リモートマシン上でコマンドを実行するためのプログラムのこと。2 つのホスト間で安全な暗号化通信を行うことを目的としています。 X11 接続、任意の TCP ポート、UNIX ドメインソケットも安全なチャンネルで転送することができます。
