## 基本

`adduser` は、`useradd` コマンドの高レベルなラッパーであり、対話的にユーザーアカウントを作成する際に便利です。多くの設定がデフォルトで行われ、ユーザーに関する情報を順次入力するプロンプトが表示されるため、初心者でも簡単に使用できます。

### 構文

```bash frame=none
# FILE のグループを GROUP に変更する
chgrp [OPTION]... GROUP FILE...
# `--reference` で各 FILE を RFILE のグループに変更する。
chgrp [OPTION]... --reference=RFILE FILE...
```

### EXAMPLES

```bash frame=none
sudo adduser ユーザー名
```

#### 使い方

##### ユーザーの作成

```bash frame=none
sudo adduser bob
```

このコマンドを実行すると対話形式のプロンプトが表示される。

```bash frame=none
Adding user `bob` ...
Adding new group `bob` (1001) ...
Adding new user `bob` (1001) with group `bob` ...
Creating home directory `/home/bob` ...
Copying files from `/etc/skel` ...
Enter new UNIX password:
Retype new UNIX password:
passwd: password updated successfully
Changing the user information for bob
Enter the new value, or press ENTER for the default
        Full Name []:
        Room Number []:
        Work Phone []:
        Home Phone []:
        Other []:
Is the information correct? [Y/n]
```

1. **パスワードの有効期限設定**: 新しいユーザーのパスワードを設定する
1. **ユーザー情報の入力**: 任意の追加情報を入力
1. **確認**: 入力内容の確認、正しければ Y を実行する

#### 作成後にユーザーをグループに追加する

`adduser` コマンド自体にはグループを追加するオプションはありませんが、作成後にユーザーをグループに追加することができます。

```bash frame=none
# ユーザー (charlie) を sudo グループに追加する
sudo adduser charlie sudo
```
