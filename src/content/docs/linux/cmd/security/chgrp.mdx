## 基本

### 構文

```bash frame=none
# FILE のグループを GROUP に変更する
chgrp [OPTION]... GROUP FILE...
# `--reference` で各 FILE を RFILE のグループに変更する。
chgrp [OPTION]... --reference=RFILE FILE...
```

### EXAMPLES

```bash frame=none
chgrp staff /u
# Change the group of /u to "staff".

chgrp -hR staff /u
# Change the group of /u and subfiles to "staff".

```
