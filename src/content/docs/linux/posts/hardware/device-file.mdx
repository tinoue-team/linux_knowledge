## /dev

Linux でハードウェア(接続機器)はファイルで管理されます。デバイスを接続すると、/dev 配下に検出されたデバイスへアクセスするためのファイル(デバイスファイル)が追加される。デバイスファイルは、デバイス(周辺機器)を操作するためのファイル。

![OSとデータの棲み分け2](src/assets/2024-09-30_8.30.57.webp)

例えば、CD/DVD ドライブのデバイスファイルは /dev/sr0 (2 台目は sr1) となる。

ディスクのデバイスファイル名は接続するインターフェースによって異なりますが、
大半で使われている SCSI (スカジー)系のディスクでは、/dev/sda (1 台目)、/dev/sdb (2 台目)、…

### ハードディスクの規格

ハードディスクには、SCSI、SATA、IDE/ATA (PATA)、USB などいくつかの企画がある。
SATA、IDE/ATA (PATA)、USB の場合はカーネル内の SCSI と同じく、/dev/sd○ として作成され、○ には 1 台目から a, b, c … と付与されていく
